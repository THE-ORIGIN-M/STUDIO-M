# 🚀 Studio M: El Ecosistema de Estilo y Tecnología

Bienvenido al laboratorio de **Studio M**. Este proyecto no es solo una tienda de dropshipping; es una plataforma digital diseñada para curar lo mejor en moda urbana y tecnología para el consumidor moderno.

## 🎯 Nuestra Misión
En Studio M, eliminamos la fricción entre la necesidad y la adquisición. Facilitamos el acceso a productos de alta rotación (gorras, apparel y gadgets) con una experiencia de usuario impecable.

## 📦 Nuestro Catálogo (The Core)
Trabajamos bajo tres pilares de producto, seleccionados por su alta demanda:

1.  **Headwear (Gorras):** El complemento definitivo. Diseños urbanos y minimalistas.
2.  **Apparel (Moda):** Prendas versátiles para hombre y mujer. Calidad y tendencia.
3.  **Tech Gadgets:** Soluciones electrónicas que simplifican el día a día.

## 🛠️ Stack Tecnológico
Studio M está construido sobre una arquitectura limpia:
- **Framework:** HTML/JS/CSS
- **Deployment:** GitHub Pages / Vercel.
- **Enfoque:** Mobile-first (optimizado para el 90% de nuestros usuarios).

## 🛡️ Compromiso de Calidad
Entendemos que la confianza es la moneda del comercio online. Por ello, Studio M garantiza:
- **Transparencia:** Seguimiento de pedidos en tiempo real.
- **Soporte:** Atención directa para resolver cualquier duda sobre tus productos.
